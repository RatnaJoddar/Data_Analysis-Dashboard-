# Online Store Data Analysis Dashboard Creation Using MS Excel
## Project Objective
An online store want to create an annual sales report for 2022. So that, they can understand their customers and grow more sales in 2023

Source: Online Store sales data (2022)
Includes:
Order details
Customer demographics (age, gender)
Sales channels
Order status
State-wise data
Product categories

Compare sales vs orders in a single chart
Identify month with highest sales & orders
Analyze gender-wise purchasing behavior
Explore order status distribution
Find top 10 states contributing to sales
Understand relationship between age & gender (orders)
Identify top sales channels
Determine highest selling category
Calculate % of total orders delivered
Dashboard Interaction
